# Countdown timer for FMI exams

* Clone
```
git clone git@github.com:gvkunchev/py-countdown.git
```
* Create venv
```
python -m venv venv
```
* Activate venv
```
.\venv\Scripts\activate
```
* Install dependencies
```
python -m pip install -r requirements.txt
```
* Run
```
python countdown/main.py
```
